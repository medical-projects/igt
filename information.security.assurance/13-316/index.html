Information Security Assurance

11-316

There is an information asset register that includes all key information, software, hardware and services

Level 1.

a.

Responsibility has been assigned for compiling and maintaining an information asset register.

Assign responsibility

Level 2.

a.

All information assets have been documented in a register that includes relevant details about each asset (ie the location of each asset, what type of information, who uses it etc).

Information Asset Register

Level 3.

a.

The asset register is maintained, updated and regularly reviewed, eg to ensure that each asset is still required and is still in use or to add new assets to the register.
